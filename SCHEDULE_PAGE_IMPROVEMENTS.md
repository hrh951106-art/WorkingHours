## ⌨️ 键盘快捷键

### 基础快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl/Cmd + C` | 复制排班 | 复制选中区域的所有排班到剪贴板 |
| `Ctrl/Cmd + V` | 粘贴排班 | 将剪贴板的排班粘贴到选中的目标位置 |
| `Delete` | 删除排班 | 删除选中区域的所有排班 |
| `Backspace` | 删除排班 | 删除选中区域的所有排班 |
| `Ctrl/Cmd + A` | 全选 | 选择所有员工和当前月份的所有日期 |
| `Esc` | 取消选择 | 清空所有选中的员工和日期 |

### 使用说明

1. **复制排班**
   - 使用鼠标选择要复制的员工（勾选复选框）
   - 点击表头的复选框选择目标日期
   - 按下 `Ctrl/Cmd + C` 复制排班到剪贴板
   - 系统会显示复制成功的提示

2. **粘贴排班**
   - 选择目标员工和目标日期
   - 按下 `Ctrl/Cmd + V` 粘贴排班
   - 系统会弹出确认对话框，显示即将粘贴的排班数量
   - 确认后执行粘贴操作

3. **删除排班**
   - 选择要删除的员工和日期
   - 按下 `Delete` 或 `Backspace`
   - 系统会显示确认对话框，显示将要删除的排班数量
   - 确认后执行删除操作

4. **全选**
   - 按下 `Ctrl/Cmd + A` 快速选择所有员工和日期
   - 适合批量操作场景

5. **取消选择**
   - 按下 `Esc` 快速清空所有选择
   - 适合重新选择场景

## 📋 剪贴板功能

### 剪贴板特性

1. **数据有效期**
   - 剪贴板数据有效期为 30 分钟
   - 超过有效期后，系统会提示"剪贴板为空或已过期"

2. **剪贴板提示**
   - 页面顶部显示剪贴板状态条
   - 当剪贴板有数据时，显示：
     - 源日期
     - 排班数量
     - 可粘贴提示

3. **智能粘贴**
   - 支持一对多：一个员工的排班粘贴到多个员工
   - 支持多对多：多个员工的排班粘贴到多个日期
   - 自动处理数据映射关系

## 🎯 操作流程示例

### 场景 1：批量排班

1. 选择 5 位员工
2. 选择 3 个工作日
3. 点击"批量排班"按钮
4. 选择班次（如：早班）
5. 确认后自动创建 15 条排班记录

### 场景 2：复制排班

1. 选择要复制的源员工和源日期
2. 按下 `Ctrl/Cmd + C` 复制
3. 选择目标员工和目标日期
4. 按下 `Ctrl/Cmd + V` 粘贴
5. 确认后完成复制

### 场景 3：快速清除排班

1. 选择要清除的员工和日期
2. 按下 `Delete` 键
3. 确认后批量删除排班

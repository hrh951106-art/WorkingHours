# 参与人配置功能说明

## 功能概述

工作流设计器中的参与人配置已经升级为弹窗模式，支持两种类型的参与人配置：

### 1. 固定人员
- 可以选择具体的用户作为审批人或抄送人
- 从用户列表中选择，支持搜索功能
- 显示用户姓名和员工编号

### 2. 组织管理者
支持多个层级的组织管理者配置：
- **选项1：申请人** - 选择提交申请的人
- **选项2：所在组织** - 选择申请人所在当前组织的管理者
- **选项3：所在组织向上一级** - 选择向上一级组织的管理者
- **选项4：所在组织向上2级** - 选择向上二级组织的管理者
- **选项5：所在组织向上3级** - 选择向上三级组织的管理者
- **选项6：所在组织向上4级** - 选择向上四级组织的管理者
- **选项7：所在组织向上5级** - 选择向上五级组织的管理者

## 使用方法

### 在工作流设计器中配置参与人

1. **打开工作流设计器**
   - 导航到：工作流定义列表 → 新增/编辑流程 → 设计器

2. **添加审批或抄送节点**
   - 点击"审批"按钮添加审批节点
   - 点击"抄送"按钮添加抄送节点

3. **配置参与人**
   - 点击已添加的审批或抄送节点
   - 在右侧配置面板中找到"审批人配置"或"抄送人配置"
   - 点击按钮打开参与人配置弹窗

4. **添加参与人**
   - **固定人员**：
     1. 选择"固定人员"类型
     2. 从下拉列表中选择具体人员
     3. 点击"添加参与人"按钮

   - **组织管理者**：
     1. 选择"组织管理者"类型
     2. 从下拉列表中选择组织层级（如：所在组织、向上1级等）
     3. 点击"添加参与人"按钮

5. **管理已添加的参与人**
   - 查看已添加的参与人列表
   - 点击删除按钮可以移除不需要的参与人
   - 可以混合添加固定人员和组织管理者

6. **保存配置**
   - 点击弹窗的"确定"按钮保存参与人配置
   - 点击配置面板的"保存配置"按钮保存节点配置

## 数据结构

参与人配置的数据结构如下：

```typescript
interface Participant {
  id?: string;                    // 唯一标识
  type: 'FIXED_USER' | 'ORG_MANAGER';  // 参与人类型
  userId?: number;                // 用户ID（固定人员）
  userName?: string;              // 用户姓名（固定人员）
  orgLevel?: number;              // 组织层级（组织管理者）
  orgLevelName?: string;          // 组织层级名称（组织管理者）
}
```

### 组织层级说明

| orgLevel | 说明 |
|----------|------|
| 0 | 申请人 |
| 1 | 所在组织 |
| 2 | 所在组织向上一级 |
| 3 | 所在组织向上2级 |
| 4 | 所在组织向上3级 |
| 5 | 所在组织向上4级 |
| 6 | 所在组织向上5级 |

## 节点显示

工作流设计器中的节点会显示参与人数量：
- **审批节点**：显示"审批人: X人"
- **抄送节点**：显示"X人"

## 特性

- ✅ 支持多选：可以添加多个参与人
- ✅ 类型混合：可以同时添加固定人员和组织管理者
- ✅ 去重检查：自动检查重复的参与人
- ✅ 实时预览：实时显示已添加的参与人数量
- ✅ 标签区分：使用不同颜色标签区分固定人员（蓝色）和组织管理者（绿色）

## 注意事项

1. 至少需要添加一个参与人才能保存配置
2. 同一用户或同一组织层级不能重复添加
3. 组织层级是基于申请人的组织关系动态计算的
4. 配置保存在节点的 `approvers`（审批人）或 `ccUsers`（抄送人）字段中

# 排班管理页面 - 键盘多选功能指南

## 🎯 Excel 风格键盘导航

现在排班管理页面支持类似 Excel 的键盘操作方式，无需鼠标即可完成所有操作！

## ⌨️ 键盘快捷键完整列表

### 导航与选择

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `↑` `↓` `←` `→` | 移动焦点 | 在表格中移动蓝色焦点框 |
| `Shift + ↑` | 向上扩展选择 | 从当前焦点向上选择到上一行 |
| `Shift + ↓` | 向下扩展选择 | 从当前焦点向下选择到下一行 |
| `Shift + ←` | 向左扩展选择 | 从当前焦点向左选择到前一列 |
| `Shift + →` | 向右扩展选择 | 从当前焦点向右选择到后一列 |
| `Space` | 切换选中 | 切换单个格子的选中状态（紫色背景） |
| `Ctrl/Cmd + A` | 全选 | 选择所有员工和当前月份的所有日期 |
| `Esc` | 取消选择 | 清空所有选择，焦点保持不变 |

### 编辑操作

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl/Cmd + C` | 复制排班 | 复制选中区域的所有排班到剪贴板 |
| `Ctrl/Cmd + V` | 粘贴排班 | 将剪贴板的排班粘贴到选中的目标位置 |
| `Delete` | 删除排班 | 删除选中区域的所有排班 |
| `Backspace` | 删除排班 | 删除选中区域的所有排班 |

## 📖 详细使用指南

### 1️⃣ 基础导航

**移动焦点**
```
步骤：
1. 打开排班管理页面（http://localhost:5173/shift/schedules）
2. 默认焦点在左上角的第一个日期上（蓝色发光边框）
3. 使用方向键在表格中移动：
   • ↑ 上移一位员工
   • ↓ 下移一位员工
   • ← 前一个日期
   • → 后一个日期
```

**视觉效果**
- 焦点单元格：**蓝色发光边框**，轻微放大
- 无排班的格子：灰色 `-` 符号
- 有排班的格子：彩色渐变卡片，显示班次名称

### 2️⃣ 单元格选择方式

#### 方式一：Space 切换（适合不连续选择）

```
操作步骤：
1. 使用方向键将焦点移动到目标单元格
2. 按下 Space 键
3. 该单元格变为**紫色背景**（已选中）
4. 再次按下 Space 键取消选中

适用场景：
• 选择不连续的多个单元格
• 精确控制每个单元格的选中状态
```

#### 方式二：范围选择（适合连续区域）

```
操作步骤：
1. 将焦点移动到选择区域的起始位置（例如：3号员工的5号）
2. 按住 Shift 键不放
3. 使用方向键扩展选择范围：
   • Shift + ↓ 向下扩展到更多员工
   • Shift + → 向右扩展到更多日期
4. 选择范围内的所有单元格变为**紫色背景**
5. 释放 Shift 键完成选择

视觉效果：
• 选择的矩形区域全部变为紫色背景
• 焦点位置显示蓝色边框
• 可以清晰地看到选择范围
```

### 3️⃣ 实际操作流程

#### 场景 1：批量复制排班（最常用）

```
目标：将周一到周三的排班复制到下周

步骤：
1. 使用方向键移动到源区域左上角（例如：3号员工的周一）
2. 按住 Shift 键
3. 按 → 键扩展到周三（选中3天）
4. 按 ↓ 键扩展到其他员工（选中多个员工）
5. 释放 Shift，确认选择区域正确（紫色背景）
6. 按 Ctrl + C 复制
7. 系统提示："已复制 X 条排班到剪贴板"
8. 使用方向键移动到目标位置（下周的周一）
9. 按住 Shift 键，用方向键选择目标区域
10. 按 Ctrl + V 粘贴
11. 确认粘贴操作

⏱️ 时间节省：相比手动勾选，效率提升 5-10 倍！
```

#### 场景 2：快速删除排班

```
目标：删除某个周末的所有排班

步骤：
1. 将焦点移动到周末第一天的某个员工
2. 按住 Shift + ↓ 选择所有员工
3. 按住 Shift + → 选择到周日
4. 整个周末区域变为紫色背景
5. 按 Delete 键
6. 确认删除对话框

✅ 优点：一次性删除多行多列，无需逐个点击
```

#### 场景 3：不连续选择

```
目标：选择某些特定的格子进行操作

步骤：
1. 移动到第一个目标位置，按 Space 选中（变紫色）
2. 移动到下一个位置，按 Space 选中
3. 重复以上步骤选择多个不连续的单元格
4. 如需取消某个选择，移动到该位置按 Space

💡 技巧：可以先用 Shift+方向键选择大范围，
   然后用 Space 取消某些不需要的格子
```

#### 场景 4：全选操作

```
目标：选择整个月的排班

步骤：
1. 按 Ctrl + A
2. 所有单元格瞬间变为紫色背景
3. 可以进行批量复制、删除等操作

⚡ 快速：相比手动逐个勾选，只需 1 秒钟
```

### 4️⃣ 高级技巧

#### 组合操作技巧

```
技巧 1：先选大范围，再精确调整
1. 按 Ctrl + A 全选
2. 用方向键移动到不需要的区域
3. 按 Space 取消选中这些格子
4. 得到精确的选择范围

技巧 2：利用 Esc 快速重选
1. 选择了一个区域后发现不对
2. 按 Esc 立即清空选择
3. 重新开始选择，焦点保持不变

技巧 3：复制粘贴的智能定位
1. 复制 2x3 的区域（2个员工 x 3天）
2. 将焦点移到目标位置
3. 按 Ctrl + V
4. 系统自动按照相同维度粘贴
```

### 常见问题

**Q: 如何快速选择整个月？**
A: 按 Ctrl + A 即可全选。

**Q: 选择错了怎么快速取消？**
A: 按 Esc 键立即清空所有选择。

**Q: 可以同时选择多个不连续的区域吗？**
A: 可以！使用 Space 键逐个选择不连续的格子。

**Q: 焦点和选中有什么区别？**
A: 焦点（蓝色边框）是当前键盘位置，选中（紫色背景）是标记要进行操作的格子。

**Q: 复制粘贴会保留格式吗？**
A: 是的！班次、时间、调整信息都会完整复制。

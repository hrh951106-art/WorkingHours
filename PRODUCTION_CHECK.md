## 🚀 部署建议

### 快速部署（Docker）
```bash
# 1. 修改环境变量
cp backend/.env.production backend/.env
cp frontend/.env.production frontend/.env

# 2. 启动服务
docker-compose up -d

# 3. 检查状态
docker-compose ps
docker-compose logs
```

### 传统部署（PM2 + Nginx）
```bash
# 1. 后端部署
cd backend
npm install
npm run build
pm2 start dist/src/main.js --name jy-backend

# 2. 前端部署
cd frontend
npm install
npm run build
# 将dist目录部署到Nginx
```

## 如何查看更新

### 1. 清除浏览器缓存
由于Vite可能缓存了旧版本的代码，请尝试以下方法：

**方法A：硬刷新（推荐）**
- Windows/Linux: `Ctrl + Shift + R` 或 `Ctrl + F5`
- Mac: `Cmd + Shift + R`

**方法B：清除缓存**
1. 打开浏览器开发者工具（F12）
2. 右键点击刷新按钮
3. 选择"清空缓存并硬性重新加载"

**方法C：无痕模式**
1. 打开新的无痕窗口
2. 访问 http://localhost:5174/

### 2. 验证更新是否生效

访问以下页面并检查列是否已更新：

**考勤工时结果页签：**
- 访问：http://localhost:5174/calculate/work-hour-results
- 选择"考勤工时结果"页签
- 检查列标题是否为：
  - ✅ 员工号
  - ✅ 员工姓名
  - ✅ 排班日期（原"计算日期"）
  - ✅ 班次
  - ✅ 出勤代码（应显示名称而非代码）
  - ✅ 开始时间
  - ✅ 结束时间
  - ✅ 实际时长（原"工作时长"）
  - ✅ 劳动力账户（新增列）
  - ✅ 状态（应为"待确认"而非"草稿"）

**精益工时结果页签：**
- 选择"精益工时结果"页签
- 两个页签的列应该完全一致

### 4. 如果仍然看不到更新

1. 检查前端控制台是否有错误：
   - 打开开发者工具（F12）
   - 查看 Console 标签页

2. 重启前端服务：
   ```bash
   # 停止当前前端服务（Ctrl+C）
   # 然后重新启动
   cd frontend
   npm run dev
   ```

3. 检查网络请求：
   - 开发者工具 > Network 标签页
   - 刷新页面
   - 查看 `/api/calculate/work-hour-results` 请求的响应数据

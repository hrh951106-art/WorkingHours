## 🔧 可能的问题和解决方案

### 问题1: Token过期（最可能）

**现象**: 后端API测试返回 `401 Unauthorized`

**原因**:
- JWT Token有时效性（通常是24小时）
- Token过期后需要重新登录

**解决方案**:
1. **清除localStorage并重新登录**
   ```javascript
   // 在浏览器控制台执行
   localStorage.clear();
   // 然后刷新页面重新登录
   ```

2. **或者在页面中点击退出登录，然后重新登录**

### 问题2: 后端服务未启动或端口错误

**检查方法**:
```bash
# 查看后端服务是否在3001端口运行
lsof -i :3001
```

**解决方案**:
```bash
cd backend
npm run start:dev
```

### 问题3: 环境变量配置问题

**检查文件**: `frontend/.env`

```bash
# 确认API_BASE_URL配置
cat frontend/.env | grep API
```

**期望值**:
```
VITE_API_BASE_URL=http://localhost:3001/api
```

### 问题4: 浏览器缓存问题

**解决方案**:
1. 清除浏览器缓存
2. 使用硬刷新: `Ctrl + Shift + R` (Windows) 或 `Cmd + Shift + R` (Mac)
3. 或者在开发者工具中禁用缓存

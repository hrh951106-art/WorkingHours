## 💡 快速上手

### 场景1：创建组织和员工

1. 点击左侧菜单 **人事管理 → 组织管理**
2. 点击 **新建组织**，创建层级结构：
   - 集团总部 → 技术部 → 后端组
3. 点击 **人事管理 → 人员管理**
4. 点击 **新建人员**，录入员工信息
5. 保存后可在列表中查看

### 场景2：排班操作

1. 点击 **排班管理 → 班次管理**
2. 点击 **新建班次**，配置：
   - 班次名称：正常班
   - 时间段：08:00-12:00, 13:30-17:30
   - 标准工时：8小时
3. 点击 **排班管理 → 排班管理**
4. 点击 **新建排班**，选择员工和班次
5. 在日历中查看排班情况

### 场景3：工时计算

1. 确保已有排班数据
2. 点击 **工时计算**
3. 选择计算日期
4. 点击 **执行计算**
5. 查看计算结果和统计信息

### 场景4：查看考勤报表

1. 点击 **考勤管理**
2. 选择日期范围
3. 查看统计卡片：
   - 出勤天数
   - 标准工时总计
   - 实际工时总计
   - 加班工时总计
4. 在表格中查看每日明细

## 🐛 常见问题

### Q1: 页面提示 "Unexpected Application Error"
**A**: 已修复！路由懒加载组件已添加Suspense包裹。

### Q2: 登录后页面空白
**A**:
1. 打开浏览器控制台 (F12)
2. 查看错误信息
3. 刷新页面 (Ctrl+R / Cmd+R)

### Q3: API请求失败
**A**:
1. 确认后端服务运行正常
2. 检查 http://localhost:3001 是否可访问
3. 查看后端日志：`tail -f backend/logs/backend.log`

### Q4: 数据未显示
**A**:
1. 先添加基础数据（组织、员工、班次）
2. 进行排班操作
3. 然后才能查看考勤和计算结果

### Q5: 如何停止服务
**A**:
```bash
cd /Users/aaron.he/Desktop/AI/JY
./stop-services.sh
```

或：
```bash
pkill -f "nest start" && pkill -f "vite"
```

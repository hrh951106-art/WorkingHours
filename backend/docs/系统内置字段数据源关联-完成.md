## 🎯 核心设计理念

### 系统内置字段（isSystem = true）

```
EmployeeInfoTabField (isSystem=1)
    ↓ dataSourceId
DataSource
    ↓ options
DataSourceOption (男、女、56个民族等)
```

**特点**：
- ✅ 直接关联数据源
- ✅ 前端禁止删除
- ✅ 前端禁止修改字段类型
- ✅ 通过查找项管理选项

### 自定义字段（isSystem = false）

```
EmployeeInfoTabField (isSystem=0, fieldType='CUSTOM')
    ↓ fieldCode
CustomField
    ↓ dataSourceId
DataSource
    ↓ options
DataSourceOption
```

**特点**：
- ✅ 通过 CustomField 管理
- ✅ 前端允许删除
- ✅ 前端允许修改
- ✅ 灵活配置

## 🔧 后端代码逻辑

### enrichFieldWithType() 方法

```typescript
private enrichFieldWithType(field: any, customFields: any[]) {
  // ✨ 系统内置字段：直接使用 field.dataSource
  if (field.isSystem) {
    return {
      ...field,
      type: field.fieldType === 'SELECT' ? 'SELECT' : 'TEXT',
      dataSource: field.dataSource || null,
    };
  }

  // 自定义字段：从 CustomField 中查找数据源
  if (field.fieldType === 'CUSTOM') {
    const customField = customFields.find((cf) => cf.code === field.fieldCode);
    return {
      ...field,
      type: customField?.type || 'TEXT',
      dataSource: customField?.dataSource || null,
    };
  }

  return field;
}
```

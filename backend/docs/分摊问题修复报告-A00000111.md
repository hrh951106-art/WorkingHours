## 注意事项

### 1. 时间戳问题
- LineShift.scheduleDate 使用 UTC 时间戳
- CalcResult.calcDate 也使用 UTC 时间戳
- 创建记录时必须使用统一的时间格式（避免时区转换问题）

### 2. 产线关联
- LineShift.lineId 必须正确关联到 ProductionLine.id
- 如果lineId为null，产线信息将无法加载

### 3. 参与分摊标志
- participateInAllocation 必须为 true
- 否则getActiveLinesForDate不会返回该产线

### 4. 软删除机制
- LineShift 使用 deletedAt 字段进行软删除
- 查询时必须添加 deletedAt: null 条件

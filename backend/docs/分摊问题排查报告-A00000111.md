## 问题根因

### ❌ 分摊规则没有配置目标

**问题详情**：
- 分摊配置 A0100001 的规则30（分摊规则1）已配置
- 规则类型：按实际工时比例分摊（PROPORTIONAL）
- **但是规则没有配置任何分摊目标**（AllocationRuleTarget 表中规则30的记录数为0）

**影响**：
- 没有分摊目标，无法计算分摊比例
- 即使有符合条件的工时记录，也无法产生分摊结果
- 这是导致员工 A00000111 在 3月15日未产生分摊结果的直接原因

## 解决方案

### 方案1：添加分摊目标（推荐）

在前端配置页面为分摊规则30添加目标：

1. **进入分摊配置页面**
   - 打开"间接工时分摊"配置
   - 找到"分摊规则1"

2. **添加分摊目标**
   - 点击"添加目标"按钮
   - 选择目标类型（如：产线、车间、工位等）
   - 设置目标对象和权重

3. **配置示例**
   ```
   目标1：
   - 目标类型：产线（LINE）
   - 目标对象：W1产线A
   - 权重：1

   目标2：
   - 目标类型：产线（LINE）
   - 目标对象：W1产线B
   - 权重：1
   ```

4. **保存并激活配置**
   - 保存规则配置
   - 确保配置状态为 ACTIVE

5. **重新执行分摊计算**
   - 使用配置 A0100001 重新执行分摊计算
   - 日期范围：2026-03-15

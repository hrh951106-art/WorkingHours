## 🚀 快速执行步骤

### 第一步：备份数据库（必做！）

```bash
pg_dump -U postgres -h localhost -d your_database_name > backup_20260401.sql
```

### 第二步：执行数据修复SQL

```bash
psql -U postgres -h localhost -d your_database_name -f backend/scripts/production-fix-complete.sql
```

**验证修复结果**:

```sql
-- 应该返回 11
SELECT COUNT(*) FROM "CustomField"
WHERE code IN (
  'gender', 'nation', 'maritalStatus', 'politicalStatus',
  'educationLevel', 'educationType', 'jobLevel',
  'employeeType', 'resignationReason', 'familyRelation', 'A05'
);

-- 应该返回 0（无重复）
SELECT code, COUNT(*) FROM "DataSource"
GROUP BY code HAVING COUNT(*) > 1;
```

### 第三步：部署代码

```bash
# 停止服务
pm2 stop jy-backend
systemctl stop nginx

# 部署后端
cp -r backend/dist/* /path/to/production/backend/dist/

# 部署前端
cp -r frontend/dist/* /path/to/production/frontend/dist/

# 重启服务
pm2 start jy-backend
systemctl start nginx
```

### 第四步：验证功能

1. 打开系统，清除浏览器缓存（Ctrl+Shift+Delete）
2. 进入"人事管理" → "新增人员"
3. 检查以下字段是否显示为下拉框：
   - ✅ 性别（显示2个选项）
   - ✅ 民族（显示8个选项）
   - ✅ 婚姻状况（显示4个选项）
   - ✅ 政治面貌（显示5个选项）
4. 填写所有必填字段，点击保存
5. ✅ 确认保存成功，无错误提示

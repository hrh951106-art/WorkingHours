# 分摊自动创建账户功能说明

## 功能概述

在执行工时分摊规则时，如果目标产线的间接设备账户不存在，系统会自动创建该账户，确保分摊能够正常执行。

## 功能特点

### 1. 智能账户名称生成

账户名称格式：`富阳工厂/{车间名称}/{产线名称}////间接设备`

示例：
- `富阳工厂/W1总装车间/L1产线////间接设备`
- `富阳工厂/W2总装车间/L3产线////间接设备`

### 2. 层级账户创建

系统会按层级自动创建账户：

1. **车间级别账户**（如果不存在）
   - 名称：`富阳工厂/{车间名称}/////间接设备`
   - 类型：WORKSHOP
   - 层级：2

2. **产线级别账户**
   - 名称：`富阳工厂/{车间名称}/{产线名称}////间接设备`
   - 类型：LINE
   - 层级：3
   - 父账户：车间级别账户

### 3. 自动设置hierarchyValues

创建的账户会自动设置正确的层级值：

```json
{
  "orgId": 1,          // 组织ID（从组织管理中获取）
  "workshopId": 6,      // 车间ID
  "lineId": 1           // 产线ID
}
```

### 4. 错误处理

- 如果创建失败，会在日志中记录详细错误信息
- 创建失败的分摊会跳过该产线，继续处理其他产线
- 不会影响整个分摊流程

## 使用场景

### 场景1：新增产线后首次分摊

当系统新增产线后，还没有对应的间接设备账户，首次执行分摊时会自动创建。

### 场景2：账户被误删除后重新分摊

如果间接设备账户被误删除（状态设为DELETED），重新执行分摊时会自动创建新账户。

### 场景3：跨车间分摊

不同车间的产线分摊时，系统会自动为每个车间创建独立的间接设备账户。

## 注意事项

1. **账户唯一性**：系统会先检查账户是否存在，避免重复创建
2. **车间账户优先**：创建产线账户前，会确保车间级别的账户存在
3. **组织信息同步**：车间和组织信息会从组织管理表中实时获取，确保数据一致性
4. **分摊不受影响**：即使某些账户创建失败，其他账户的分摊仍会继续执行

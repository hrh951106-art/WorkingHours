# G02分摊规则修复报告

## 问题现象
使用G02分摊规则执行分摊操作后，没有生成任何分摊结果。

## 根本原因

### 1. 代码逻辑错误
在 `allocation.service.ts` 的 `getLineIndirectAccount` 方法中，使用了错误的字段来构建间接设备账户名称。

**错误代码：**
```typescript
const accountName = `富阳工厂/W1总装车间/${line.orgName}////间接设备`;
```

这会生成：`富阳工厂/W1总装车间/富阳工厂////间接设备`

**正确代码：**
```typescript
const accountName = `富阳工厂/W1总装车间/${line.name}////间接设备`;
```

这会生成：`富阳工厂/W1总装车间/L1产线////间接设备`

### 2. 数据缺失
部分产线缺少对应的间接设备账户，导致分摊逻辑在检查账户时返回null，跳过分摊。
